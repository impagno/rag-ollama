This project uses Ollama and SQLite to aswer some questions about Artemis 2 mission.

To install Ollama locally, please run the follow commands on PS:<br/>
<i>irm https://ollama.com/install.ps1 | iex</i>

To pull the Ollama embeddings model:<br/>
<i>ollama pull bge-large:335m</i>

To pull the local Llama3 image:<br/>
<i>ollama pull llama3.2:3b</i>
